◆ timerSetInputCaptureFilter()

TIMER_RETURN_CODE_t timerSetInputCaptureFilter ( TIM_TypeDef *  tim,
TIMER_CHANNEL_NUM_t  channel,
TIMER_IC_FILTER_t  filter 
)

This function defines the frequency used to sample TI1 input and the length of the digital filter applied to TI1. The digital filter is made of an event counter in which N consecutive events are needed to validate a transition on the output.

FCKINT is the internal clock / FDTS = Is the clock frequency for all filters. It is derived from FCKINT

Parameters
*tim: Pointer to the desired timer
channel: Selects the input channel of which the input signal shall be filtered
filter: Filter setting (frequency)
