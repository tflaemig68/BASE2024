void systickSetMicros ( uint32_t *  timer,
uint32_t  micros 
)
Deprecated:
Sets the tick interval of the software timer in microseconds.
Parameters
*timer: Pointer to the software timer
micros: Tick time of the software timer in microseconds
Note
This function is formally 100% compatible with the newly provided function systickSetTicktime(), i.e. the code is identical to the new function systickSetTicktime(). The reason for the replacement is as follows: The name systickSetMicros() might be confusing since the user might expect to get a ticktime of a multiple of one microsecond. However, depending on the setting of the divider used in function systickInit() the real ticktime might be also a multiple of 10 or 100 microseconds.
