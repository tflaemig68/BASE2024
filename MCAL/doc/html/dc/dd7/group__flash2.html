void flashConfigWaitStates ( uint16_t  sysclk)

mcalFlashCtrl.c

Created on: 03.07.2021 Author: Ralf

Configures the number of wait states (WS) when accessing the internal Flash memory.

Parameters
sysclk: The number of wait states depends on the MCU clock and the supply voltage. However, this function doesn't take care of the supply voltage.
Note
The number of WS depends immediately from the supply voltage and the selected MCU clock. This function assumes that the supply voltage is in the range of 2.7 to 3.6V. Other voltages must be handled separately. This function enables all caches by default.
