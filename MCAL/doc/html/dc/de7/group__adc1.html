◆ adcSetChannelSequence()

ADC_RETURN_CODE_t adcSetChannelSequence ( ADC_TypeDef *  adc,
ADC_CHANNEL_t *  chnList,
size_t  seqLen 
)

Defines a list of channels which shall be monitored within a sequence.

A sequence can use up to 16 channels of the desired ADC. This function returns an error code if the size of the list exceeds 16 entries!

Parameters
*adc: Pointer to the ADC component
*chnListA pointer to an array which provides up to 16 ADC input channels
seqLen: Length of the sequence of ADC channels


Affected register and bit(s)

Register Bit name Bit(s)
SQR1/SQR2/SQR3 Assigns all channels in the given order and sets the length of the sequence Depends on the registers
Note
The sequence starts always at position 0 in the sequence list.
