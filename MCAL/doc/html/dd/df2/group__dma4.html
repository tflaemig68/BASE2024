◆ dmacDisableFifoMode()

DMAC_RETURN_CODE_t dmacDisableFifoMode ( DMA_Stream_TypeDef *  stream)

Disables the DMA FIFO direct mode.

Parameters
*stream: Pointer to the DMA stream
Returns
DMAC_RETURN_CODE_t
Note
Consider that the DMDIS bit must be '1' to disable the direct mode. Direct mode is forbidden if the data flow-direction is
MEM_2_MEM.
Disabling the DMA FIFO mode implicetely enables the 'Direct mode' (DMDIS bit in FCR is then reset to 0).
