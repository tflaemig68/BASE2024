◆ gpioSelectAltFunc()

GPIO_RETURN_CODE_t gpioSelectAltFunc ( GPIO_TypeDef *  port,
PIN_NUM_t  pin,
ALT_FUNC_t  af 
)

Selects the alternate function (AF) of the GPIO pin. Verifies the integrity of the port, pin and the alternate function parameter.

Note
AFR is a two-element array of the GPIO_TypeDef structure. Since AFR[1] starts at bit 0 we must subtract '8' from the pin number. Finally, the result must be multiplied by '4' since four bits must be moved to the left.
Parameters
*port: Pointer to the GPIO port
pin: Number of the GPIO pin
af: Number of the alternative function
Returns
GPIO_RETURN_CODE_t
