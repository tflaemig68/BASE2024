DMAC_RETURN_CODE_t dmacDisableStream ( DMA_Stream_TypeDef *  stream)

Disables the stream. It verifies the desired stream parameter. This function checks the EN bit and returns only when the stream is disabled (EN = 0)!

Parameters
*stream: Pointer to the DMA stream
Returns
DMAC_RETURN_CODE_t
Note
It is required to disable a DMA stream BEFORE writing new settings to the registers.
